# Vrinda Store Sales Performance Analysis (2022)

## Project Overview
An Excel-based order-level sales performance analysis project focused on sales trends, customer behavior, order fulfillment status, regional contributions, and sales channel performance, supported by dashboard visualization.

## Business Objective
To analyze order-level sales data and identify trends, performance gaps, and improvement opportunities.

## Dataset
- Order-level sales data (2022)
- Each row represents an individual order record

## Data Cleaning and Processing
- Standardized text values for city and state names
- Created derived columns such as Month and Age Group using Excel functions
- Ensured consistent data formats for accurate analysis

## Tools Used
- Microsoft Excel
- Pivot Tables and Charts
- Dashboard

## Analysis Performed
- Monthly sales and order trend analysis
- Customer segmentation by age and gender
- Order status distribution analysis
- State-wise sales performance
- Sales channel performance analysis

## Dashboard
An interactive Excel dashboard was created to visualize key performance metrics, trends, and comparisons, enabling easy interpretation of sales performance and business insights.

## Key Insights
- Sales peak in March and decline toward year-end
- Women and adult customers contribute the majority of orders
- Amazon and Flipkart are the top-performing channels
- Around 8% of orders are cancelled or returned
- Sales are concentrated in a limited number of states

## Next Steps / Business Implications
- Focus inventory planning and marketing efforts during January to March to maximize peak-season sales.
- Use targeted discounts and campaigns during April–July and September–December to boost low-demand periods.
- Target women customers with personalized offers to increase revenue.
- Strengthen engagement with the adult age group to drive repeat purchases.
- Prioritize Amazon and Flipkart for order volume while promoting high-value products on premium channels.
- Maintain high delivery performance to sustain the 92% order success rate.
- Improve sales penetration in low-performing regions while strengthening presence in top-performing states.
